# IntegralCat
Компьютерная 2D-игра с элементами жанра roguelike, где после сражений игрок должен решать математические (или иные образовательные) задачи, чтобы получить ресурсы, оружие или продвинуться дальше, благодаря чему человек может играть и учиться/повторять материал одновременно.
